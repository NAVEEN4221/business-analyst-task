# Business Analyst Frontend & Backend Task

## Overview
This project demonstrates a basic user authentication flow designed from a Business Analyst perspective.

## Features
- Login / Sign-up page
- Home dashboard page
- Profile page
- Responsive frontend design

## Backend Logic
- User authentication handled using Firebase Authentication
- Email/Password based login system
- No custom backend code (low-code approach)

## User Flow
1. User signs up or logs in
2. User is redirected to Home page
3. User can view Profile details
4. User can log out

## Tools Used
- HTML, CSS
- Firebase Authentication
- GitHub for version control

## Future Enhancements
- Database integration for extended profile details
- Role-based access
- Analytics dashboard
